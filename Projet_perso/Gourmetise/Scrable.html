Projet : WinScrabble

Reproduction du jeu de Scrabble en application Windows

Développé en C# sous Visual Studio

Finalisé en février 2024

WinScrabble est une application qui reproduit le célèbre jeu de Scrabble. Ce projet a été développé en C# avec Windows Forms sur Visual Studio. Le jeu propose une interface, une gestion complète des lettres, des scores, et un mode multijoueur local à 2 joueurs.

Fonctionnalités du Jeu

  • Interface : Interface 2 joueurs avec les scores affichées la lettre à rentrée et les tours
  • Gestion des lettres : Pioche aléatoire, lettres limitées, remplacement
  • Multijoueur : Mode à 2 joueurs en local avec alternance de tours
  • Scores : Calcul automatique des points
  • Validation : Vérification des mots via dictionnaire intégré

Stack Technique


  • Langage : C# (.NET Framework)
  • IDE : Visual Studio
  • GUI : Windows Forms
  • Versioning : GitHub avec release disponible
